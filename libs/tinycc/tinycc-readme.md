TinyCC (TCC) — компактный компилятор C.

Использование в TypeOS:
1. Компиляция C‑кода «на лету» из Type‑скриптов.
2. Выполнение низкоуровневых операций (криптография, работа с памятью).

Как подключить:
- Добавьте `libtcc.so` в `libs/tinycc/`.
- В `AndroidManifest.xml` укажите путь к библиотеке.
- В Java: используйте JNI для вызова функций из `libtcc.so`.
